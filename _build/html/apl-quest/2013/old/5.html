Identity Crisis#

Problem: An Identity Matrix is a square matrix (table) of 0 with 1’s in the main diagonal. Write an APL dfn which produces an n×n identity matrix.

Video: https://youtu.be/vVaZ3wEdmpQ Code: https://github.com/abrudz/apl_quest/blob/main/2013/5.apl

  6. Becuase the horizontal and the vertical parts of this triangle are equal to each other( the same index is used on both sides of the outer product). The diagnals length is a multiple of the square root of 2.
